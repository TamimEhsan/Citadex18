## Citadex Web
Just a simple portal for showing notices, classes, files etc. in a single place. This app was built using React and Google Sheets API.

## Structure
```
- docs
    - // react static files

- src
    - route
        - Drive
        - Notice
        - Classes
        - Files
    - App.js
    - index.js
    - index.css
    - config.json

- public
    - index.html
    - favicon
```
The `config.json` contains some necessary keys.
